## 接頭辞一覧

| 接頭辞 | いつ使うか | 例 |
| -------- | ----------- | ----- |
| `add` | 新しいファイル・フォルダ・機能を追加するとき | `add: ユーザー一覧ページを追加` |
| `change` | 既存のファイルや設定を**意図的に**変更するとき（バグ修正・新規追加以外） | `change: READMEの構成を整理` |
| `feature` | 新しい**規則・方針・仕様**を定義するとき | `feature: コミットメッセージの命名規則を追加 or 新機能~~を追加` |
| `fix` | バグや誤りを修正するとき | `fix: リンク切れを修正` |
| `remove` | ファイル・フォルダ・機能を削除するとき | `remove: 未使用の画像フォルダを削除` |
| `rename` | ファイル・フォルダの名前を変更するとき | `rename: utils を helpers に変更` |
| `docs` | ドキュメント（README など）のみを更新するとき | `docs: 命名規則の表を追加` |
| `test` | テストの追加・修正をするとき | `test: ログイン処理のテストを追加` |
